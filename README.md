# Crwn Clothing

Crwn Clothing is an e-commerce app built with ReactJS. It's part of the course Complete React Developer from Zero To Mastery Academy.

## Technologies

- Typescript
- Front-end with ReactJS using hooks
- Redux and Redux-Saga (state management)
- SASS and Styled Components
- Stripe API (payment)
- Back-end with Firebase

## Features

- Account creation or sign up with Google
- Responsive design
- Skeleton loading
- Add/remove itens from cart
- Payment with Strip in test mode
